## Defects surfaced + fixed
**D19 — HIGH — Production learning accuracy is not yet certified.**  
Symptom: EP05 returns a valid TimingMap using deterministic lyric distribution, not Demucs vocal isolation plus WhisperX forced alignment.  
Root cause: Model distribution and real ML runtime are still external blockers; local implementation uses `deterministic-aligner` metadata.  
Latency: Known since EP05 was originally blocked by model/runtime packaging. Unit tests did not catch it because they validate shape and integration, not musical alignment quality.  
Evidence: `python-sidecar/lyricue_sidecar/learning.py` result diagnostics include `alignmentMode: "deterministic"`.  
Fix proposal: Add Demucs/WhisperX model-backed alignment behind the same `learn_song` stage contract, then run a public-domain audio QA pass with word timing tolerance checks.  
Fix status: Proposed.

**D20 — MEDIUM — EP13 bundle container is not ZIP yet.**  
Symptom: `.lcbundle` export/import works as a deterministic JSON envelope, not the ZIP container described in the epic.  
Root cause: Core integrity/import contract was implemented first without adding a ZIP dependency or custom ZIP writer.  
Latency: New in this pass. Tests validate integrity and import behavior but not ZIP compatibility.  
Evidence: `packages/core/src/library/library-manager.ts` encodes/decodes the bundle with `TextEncoder`/`TextDecoder`.  
Fix proposal: Replace the envelope serializer with a ZIP container containing `manifest.json`, `timing.json`, `show.json`, and `arrangements.json`, while preserving existing manifest hashes.  
Fix status: Proposed.

**D21 — HIGH — EP14 infrastructure is behavior-tested but not provisioned.**  
Symptom: Worker core is implemented and tested locally, but no idempotent Cloudflare setup script, GitHub mirror, signing-key upload, or dashboard screenshot evidence exists.  
Root cause: Those stories require Cloudflare/GitHub credentials and organization-specific deployment choices.  
Latency: Known external environment gap from handoff.  
Evidence: `docs/library-setup.md` lists manual setup caveats.  
Fix proposal: Once Cloudflare account details are available, implement the setup script and run a real deployed Worker QA pass.  
Fix status: Proposed.
